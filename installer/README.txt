Z-CLEANER v1.0.0
Nettoyeur système professionnel multiplateforme

INSTALLATION :
1. Exécutez Z-Cleaner-Setup-1.0.0.exe
2. Suivez les instructions de l'assistant d'installation
3. Acceptez le contrat de licence
4. Choisissez le dossier d'installation
5. Cliquez sur "Installer"

UTILISATION :
Lancez Z-Cleaner depuis le menu Démarrer ou le raccourci du bureau.

FONCTIONNALITÉS :
- Nettoyage des fichiers temporaires
- Analyse des fichiers volumineux
- Optimisation des performances
- Détection de malwares
- Gestion des programmes au démarrage

SUPPORT :
Pour obtenir de l'aide, visitez : https://github.com/elykik/z-cleaner

DÉSINSTALLATION :
Utilisez "Ajouter ou supprimer des programmes" dans le Panneau de configuration.

POLITIQUE DE CONFIDENTIALITÉ :
Z-Cleaner ne collecte pas vos données personnelles.
Toutes les opérations se font localement sur votre ordinateur.

---
Merci d'utiliser Z-Cleaner !
