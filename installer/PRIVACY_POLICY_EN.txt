PRIVACY POLICY

Last Updated: November 2025

================================================================================
1. INTRODUCTION
================================================================================

Z-Cleaner (hereinafter "the Application") is committed to protecting your 
privacy and providing you with a transparent and secure user experience. This 
Privacy Policy explains how we handle your personal data.

================================================================================
2. DATA COLLECTED
================================================================================

2.1 DATA COLLECTED LOCALLY

The Application collects and processes the following data ONLY on your local 
computer:

- Temporary files (Temp, Cache)
- Log files
- Application cache files
- Disk space information
- Large file names and sizes
- Startup program information

IMPORTANT: None of this data is sent to our servers.

2.2 DATA NOT COLLECTED

The Application does NOT collect:
- Your personal information (name, email, address, etc.)
- Your personal files or documents
- Your passwords or sensitive information
- Your browsing history
- Your location data
- Your contacts or communication information

================================================================================
3. DATA PROCESSING
================================================================================

3.1 LOCAL PROCESSING

All Application operations are performed locally on your computer:

- File analysis
- Temporary file detection
- File cleaning
- Disk space analysis
- Malware detection

3.2 OPTIONAL MALWARE ANALYSIS

The Application offers an OPTIONAL malware scanning feature that uses the 
Cloudmersive API:

- This feature is DISABLED by default
- You must explicitly enable it in settings
- You must provide a valid Cloudmersive API key
- Only files you select are sent to Cloudmersive
- Cloudmersive has its own privacy policy

================================================================================
4. DATA SENT TO THIRD PARTIES
================================================================================

4.1 CLOUDMERSIVE (OPTIONAL)

If you enable malware scanning with Cloudmersive:

- Only files you select are sent
- Data is sent via HTTPS (encrypted)
- Cloudmersive processes this data according to its privacy policy
- See https://www.cloudmersive.com/privacy for more information

4.2 OTHER THIRD PARTIES

The Application does not share your data with any other third parties.

================================================================================
5. DATA STORAGE
================================================================================

5.1 LOCAL STORAGE

The following data is stored locally on your computer:

- Application settings
- Cleaning history
- User preferences (language, theme)
- Cloudmersive API key (if provided)

5.2 STORAGE SECURITY

- Data is stored in standard Windows directories
- Sensitive data (API keys) are stored in the .env file
- You are responsible for your computer's security

================================================================================
6. USER RIGHTS
================================================================================

6.1 DATA ACCESS

You have the right to access your personal data stored by the Application. 
This data is stored locally on your computer.

6.2 DATA DELETION

You can delete all Application data by:
- Uninstalling the Application
- Manually deleting configuration files
- Resetting settings in the Application

6.3 DATA PORTABILITY

Your data is stored locally and you can export or transfer it at any time.

================================================================================
7. DATA SECURITY
================================================================================

7.1 SECURITY MEASURES

The Application implements the following security measures:

- Local data processing (no network transmission)
- Secure file deletion
- File validation before deletion
- Safe mode for testing without deleting
- HTTPS encryption for optional communications

7.2 RESPONSIBILITY

You are responsible for:
- Your computer's security
- Backing up your important data
- Responsible use of the Application
- Protecting your Cloudmersive API key

================================================================================
8. DATA RETENTION
================================================================================

8.1 RETENTION PERIOD

- Data is retained as long as the Application is installed
- Cleaning history is retained locally
- Settings are retained until uninstallation
- You can manually delete history

8.2 DELETION AFTER UNINSTALLATION

Upon uninstallation:
- Application files are deleted
- Configuration data may remain (depending on settings)
- You can manually delete remaining files

================================================================================

The Application does NOT use:
- Cookies

- Web beacons
- Similar tracking technologies
