POLITIQUE DE CONFIDENTIALITÉ

Dernière mise à jour : Novembre 2025

================================================================================
1. INTRODUCTION
================================================================================

Z-Cleaner (ci-après "l'Application") s'engage à protéger votre vie privée et à 
vous fournir une expérience utilisateur transparente et sécurisée. Cette 
Politique de Confidentialité explique comment nous traitons vos données 
personnelles.

================================================================================
2. DONNÉES COLLECTÉES
================================================================================

2.1 DONNÉES COLLECTÉES LOCALEMENT

L'Application collecte et traite les données suivantes UNIQUEMENT sur votre 
ordinateur local :

- Fichiers temporaires (Temp, Cache)
- Fichiers journaux (Logs)
- Fichiers de cache d'application
- Informations sur l'espace disque
- Noms et tailles de fichiers volumineux
- Informations sur les programmes au démarrage

IMPORTANT : Aucune de ces données n'est envoyée à nos serveurs.

2.2 DONNÉES NON COLLECTÉES

L'Application ne collecte PAS :
- Vos données personnelles (nom, email, adresse, etc.)
- Vos fichiers personnels ou documents
- Vos mots de passe ou informations sensibles
- Vos historiques de navigation
- Vos données de localisation
- Vos contacts ou informations de communication

================================================================================
3. TRAITEMENT DES DONNÉES
================================================================================

3.1 TRAITEMENT LOCAL

Toutes les opérations de l'Application sont effectuées localement sur votre 
ordinateur :

- Analyse des fichiers
- Détection des fichiers temporaires
- Nettoyage des fichiers
- Analyse de l'espace disque
- Détection de malwares

3.2 ANALYSE OPTIONNELLE DE MALWARES

L'Application offre une fonctionnalité OPTIONNELLE de scan de malwares qui 
utilise l'API Cloudmersive :

- Cette fonctionnalité est DÉSACTIVÉE par défaut
- Vous devez l'activer explicitement dans les paramètres
- Vous devez fournir une clé API Cloudmersive valide
- Seuls les fichiers que vous sélectionnez sont envoyés à Cloudmersive
- Cloudmersive dispose de sa propre politique de confidentialité

================================================================================
4. DONNÉES ENVOYÉES À DES TIERS
================================================================================

4.1 CLOUDMERSIVE (OPTIONNEL)

Si vous activez le scan de malwares avec Cloudmersive :

- Seuls les fichiers que vous sélectionnez sont envoyés
- Les données sont envoyées via HTTPS (chiffré)
- Cloudmersive traite ces données selon sa politique de confidentialité
- Consultez https://www.cloudmersive.com/privacy pour plus d'informations

4.2 AUTRES TIERS

L'Application ne partage vos données avec aucun autre tiers.

================================================================================
5. STOCKAGE DES DONNÉES
================================================================================

5.1 STOCKAGE LOCAL

Les données suivantes sont stockées localement sur votre ordinateur :

- Paramètres de l'Application
- Historique de nettoyage
- Préférences utilisateur (langue, thème)
- Clé API Cloudmersive (si fournie)

5.2 SÉCURITÉ DU STOCKAGE

- Les données sont stockées dans les répertoires standards de Windows
- Les données sensibles (clés API) sont stockées dans le fichier .env
- Vous êtes responsable de la sécurité de votre ordinateur

================================================================================
6. DROITS DE L'UTILISATEUR
================================================================================

6.1 ACCÈS AUX DONNÉES

Vous avez le droit d'accéder à vos données personnelles stockées par 
l'Application. Ces données sont stockées localement sur votre ordinateur.

6.2 SUPPRESSION DES DONNÉES

Vous pouvez supprimer toutes les données de l'Application en :
- Désinstallant l'Application
- Supprimant manuellement les fichiers de configuration
- Réinitialisant les paramètres dans l'Application

6.3 PORTABILITÉ DES DONNÉES

Vos données sont stockées localement et vous pouvez les exporter ou les 
transférer à tout moment.

================================================================================
7. SÉCURITÉ DES DONNÉES
================================================================================

7.1 MESURES DE SÉCURITÉ

L'Application implémente les mesures de sécurité suivantes :

- Traitement local des données (pas de transmission réseau)
- Suppression sécurisée des fichiers
- Validation des fichiers avant suppression
- Mode sûr pour tester sans supprimer
- Chiffrement HTTPS pour les communications optionnelles

7.2 RESPONSABILITÉ

Vous êtes responsable de :
- La sécurité de votre ordinateur
- La sauvegarde de vos données importantes
- L'utilisation responsable de l'Application
- La protection de votre clé API Cloudmersive

================================================================================
8. CONSERVATION DES DONNÉES
================================================================================

8.1 DURÉE DE CONSERVATION

- Les données sont conservées tant que l'Application est installée
- L'historique de nettoyage est conservé localement
- Les paramètres sont conservés jusqu'à la désinstallation
- Vous pouvez supprimer l'historique manuellement

8.2 SUPPRESSION APRÈS DÉSINSTALLATION

Lors de la désinstallation :
- Les fichiers de l'Application sont supprimés
- Les données de configuration peuvent rester (selon les paramètres)
- Vous pouvez supprimer manuellement les fichiers restants

================================================================================

- De cookies

- De technologies de suivi similaires
